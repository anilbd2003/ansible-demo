Congratulation!!! You have successfully deployed application on web server using ansible automation tool. Please see steps below for your project.

  1. Create EC2 instance as ansible master.
  2. Create key pairs on the ansible-machine "ssh-keygen -t rsa -b 2048"
  3. Create EC2 instance as ansible host.
  4. Copy public key from ansible master to ansible host.
  5. Test ssh connection (passwordless authentication)
  6. Install ansible on ansible-machine "sudo amazon-linux-extras install ansible2 -y"
  7. Create inventory file
  8. Create playbook
  9. Create ansible.cfg file
  10. Run the ansible playbook
